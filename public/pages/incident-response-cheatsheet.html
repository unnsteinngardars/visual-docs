🔌01 · Immediate Isolation
Disable NIC or pull cable — do NOT shutdown (preserve volatile memory)
Quarantine in firewall/NAC — block all except IR team source IP
Disable host AD account: Disable-ADAccount -Identity HOSTNAME$
Reset KRBTGT password only if Golden Ticket suspected (twice, 10 hr apart)
Snapshot VM before any action if virtual — preserve disk state
Notify stakeholders, open incident ticket, start chain of custody log
⚡02 · Volatile Data — Capture First
Running processes + paths
Get-Process | Select Id,Name,Path,Company | Sort Name
Network connections
netstat -anob # -b = owning process
Active sessions / shares
net session; net use; net share
Logged-on users
quser; query session
DNS cache (C2 beacon clue)
ipconfig /displaydns | findstr "Record Name"
Scheduled tasks
schtasks /query /fo LIST /v | findstr "TaskName Run"

🏰03 · AD Recon Detection
Script block logging (EID 4104)
Get-WinEvent -LogName "Microsoft-Windows-PowerShell/Operational"
LDAP recon (EID 1644 on DC)
# HKLM\...\NTDS\Diagnostics "15 Field Engineering" = 5
New domain admins
Get-ADGroupMember "Domain Admins" | Select Name,SamAccountName,WhenCreated
Recently created accounts (7d)
Get-ADUser -Filter * -Properties WhenCreated | Where {$_.WhenCreated -gt (Get-Date).AddDays(-7)}

🔍05 · PowerShell Hunt Queries
Encoded command detection
Get-WinEvent -LogName "Microsoft-Windows-PowerShell/Operational" | Where {$_.Message -like "*-EncodedCommand*"}
Suspicious parent processes
# Sysmon EID 1 ParentImage=winword|excel|outlook
LOLBins in process creation
Get-WinEvent -Id 4688 | Where {$_.Message -match "certutil|mshta|wscript|regsvr32|rundll32|bitsadmin"}
Unusual listening ports
netstat -anob | Select-String LISTEN
Files written in last 24h
Get-ChildItem C:\ -Recurse -Force -ErrorAction SilentlyContinue | Where LastWriteTime -gt (Get-Date).AddHours(-24)
🧹06 · Eradication & Recovery
Remove malicious tasks, services, registry keys, WMI subscriptions
Disable / delete all rogue accounts — audit group memberships
Revoke all Kerberos tickets: klist purge on affected hosts
Reset passwords for ALL accounts that logged on to compromised host
Review service account passwords; remove unnecessary SPNs
Patch exploited vulnerability — confirm CVE ID before closing
Re-image from known-good baseline; restore from pre-incident backup
Validate detection rules; tune SIEM/EDR; conduct post-incident review
